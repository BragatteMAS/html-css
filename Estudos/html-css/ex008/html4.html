Muitas Tags estão obsoletas!

Segue lista de principais tags que não devem mais ser usadas no desenvolvimento com HTML
Lista de TAGS para NÃO usar

applet => Use embed or object instead
acronym => Use abbr instead
dir => Use ul instead
frame or frameset or noframes =>Use iframe and CSS instead
isindex => Use an explicit form and text field combination instead
listing or xmp => Use pre and code instead
noembed => Use object instead
plaintext => Use the "text/plain" MIME type instead
basefont/big/blink/center/font/marquee/s/spacer/strike/tt/u => Use CSS instead.
